66 # This object maintains the state of a virtual text screen as a
67 # rectangluar array. This maintains a virtual cursor position and handles
68 # scrolling as characters are added. This supports most of the methods needed
69 # by an ANSI text screen. Row and column indexes are 1-based (not zero-based,
70 # like arrays).
71 #
72 # Characters are represented internally using unicode. Methods that accept
73 # input characters, when passed 'bytes' (which in Python 2 is equivalent to
74 # 'str'), convert them from the encoding specified in the 'encoding'
75 # parameter to the constructor. Methods that return screen contents return
76 # unicode strings, with the exception of __str__() under Python 2. Passing
77 # ``encoding=None`` limits the API to only accept unicode input, so passing
78 # bytes in will raise :exc:`TypeError`.
79 #
