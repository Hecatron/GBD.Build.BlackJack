Detailed Description

Works like spawn, but accepts and returns unicode strings.

Extra parameters:

:param encoding: The encoding to use for communications (default: 'utf-8') :param errors: How to handle encoding/decoding errors; one of 'strict' (the default), 'ignore', or 'replace', as described for :meth:~bytes.decode and :meth:~str.encode.
